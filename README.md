# 手写qiankun微应用

#### 介绍
手写qiankun微应用，探索qiankun微应用加载原理，感谢bilibili的鹏周老师,简单将老师讲的内容敲了一遍。
 

#### 使用说明

1.  main-app主应用，vue2-app 子应用
2.  进入两个应用中分别安装依赖包: npm i
3. main-app的入口文件处可切换qiankun与手写qiankun使用方式，手写实现放在main-app的micro-fe文件夹中
4. 文件中都有详细注释，在此就不多说了
